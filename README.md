# Wave Interference Simulator

This Python project simulates wave interference on a 2D surface.  
The user can input:

- Wavelength
- Amplitude
- Size of the plane
- Coordinates of two wave sources

The program calculates the resulting wave pattern and displays a 3D plot of the waves, showing constructive and destructive interference.

## Requirements

- Python 3.x
- numpy
- matplotlib
